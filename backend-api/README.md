# Backend - Tienda con NestJS

## Descripción

Este proyecto es el **backend** de una tienda de productos, construido con **NestJS** y **TypeScript**.  
Proporciona una API REST para la gestión de productos, usuarios y pedidos, así como integración con pagos mediante tarjeta de crédito.

---

## Características

- Listado de productos
- Creación de órdenes, transacciones, usuarios y direcciones
- Procesamiento de pagos con tarjeta de crédito
- API documentada con Postman
- Pruebas unitarias con Jest

---

## Tecnologías

- **Backend:** NestJS, Node.js, TypeScript
- **Base de datos:** PostgreSQL
- **Pruebas:** Jest
- **Documentación y pruebas de API:** Postman

---

## Instalación

```bash
# Clonar el repositorio
git clone git@github.com:Jaac25/The-Shop.git

# Ir al directorio del backend
cd backend

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run start:dev
```
